# ue5-treasurehunt/Solidity

## Quick Start
- `npm install`
- `npx hardhat test`

## Deployment
- `npx hardhat run --network goerli scripts\deploy_GamePrize.js`

## Verification
- `npx hardhat verify --network <network> <contract address>`
